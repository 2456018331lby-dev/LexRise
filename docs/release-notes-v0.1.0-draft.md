# LexRise v0.1.0

## Highlights

- Offline-first vocabulary learning for CET / exam prep
- Import word books from CSV and TXT
- Dual study modes: new words and spaced review
- Local learning progress with reminder settings
- Built-in starter book plus custom word-book switching

## Notes

- No account system in v0.1.0
- No cloud sync in v0.1.0
- Designed for fast local use and iterative open-source development
